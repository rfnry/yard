# Knowledge — Practice Reference

Reference material the lawyer or paralegal may need cited during
investigative lookups. Organized by area; drill into a subdirectory's
`_index.md` for the file list inside that area.

- [public-records/_index.md](public-records/_index.md) — what each
  public-records source actually contains, gaps, refresh cadence
- [evidence/_index.md](evidence/_index.md) — chain-of-custody and
  admissibility notes for the records this assistant pulls
- [firm/_index.md](firm/_index.md) — internal practice policies
  (retention, escalation, conflict checks)

If the topic isn't in the indices, try `Grep` against `knowledge/` —
prose is searchable.

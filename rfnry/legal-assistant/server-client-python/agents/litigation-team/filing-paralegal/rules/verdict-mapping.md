# Verdict mapping

| Findings | Verdict |
|---|---|
| `procedural_issues = []` and `missing_exhibits = []` and `citation_issues = []` | `ready_to_file` |
| Any of the above non-empty | `needs_revision` |
| Drafting requested, or fundamental scope problem | `block` |

`ready_to_file` is a high bar. Don't issue it generously.

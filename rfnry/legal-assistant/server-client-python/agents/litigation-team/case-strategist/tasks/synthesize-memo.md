# Synthesize Memo

The primary task: receive an investigation request (fresh from a
lawyer, or pre-classified by intake-clerk), delegate the lookup work
to records-investigator, and synthesize a closing memo for the lawyer.

A good outcome:

- The memo's three sections are populated honestly. No fact appears
  in **What we have** that isn't in the records-investigator's
  report this turn.
- **What we don't have** lists real gaps, not hand-waves. If the
  records-investigator returned everything the skill called for, this
  section is empty.
- **Suggested next moves** is 2 to 4 specific fact-finding actions,
  each tied to a tool the records-investigator can run.
- No strategy / opinion sneaks in.

A bad outcome:

- Fabricated facts to round out the memo.
- A "Suggested next moves" full of strategy verbs (consider,
  evaluate, weigh, push).
- An empty memo when the records-investigator did return findings —
  that means you didn't read the delegate response carefully.
- Calling the public-records tools yourself (you don't have them).

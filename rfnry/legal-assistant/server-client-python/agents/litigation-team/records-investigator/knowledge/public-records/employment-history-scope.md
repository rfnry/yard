# Employment History Scope

`EmploymentHistory` is **declared** employment — what the subject
told a credit bureau, a background-check vendor, or a public
filing (e.g. campaign-finance disclosures, court filings).

What it is **not:**

- W-2 / 1099 filings — those are IRS records, not on file.
- Real-time employer payroll — not on file.
- Self-reported social-media employment — not on file unless it
  showed up in a vendored background check.

Each entry has: employer name, declared role, declared start and
(optional) end dates, source (which filing/vendor returned it).

When the subject's stated employment differs from what's declared
in this source, that discrepancy is the finding. Quote both, cite
both, and let the lawyer decide what it means.

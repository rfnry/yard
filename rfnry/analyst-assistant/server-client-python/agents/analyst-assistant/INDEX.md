# Map

- `rules/` — what every research note must do
- `skills/` — reusable lookup procedures (pull a snapshot, scan a sector)
- `tools/` — declarative HTTP tools against the market-data backend
- `knowledge/` — research reference (data-source coverage,
  citation methodology, shared-vs-per-engagement notes); per-client
  files land at the top level alongside `_index.md`
- `tasks/` — three task surfaces:
  - `market-scan` — quick pulse on a ticker
  - `competitor-profile` — depth on one company
  - `weekly-summary` — multi-ticker summary for the client's portfolio

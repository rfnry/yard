# Filings

The `Filings` tool covers:

- **US:** SEC EDGAR (10-K, 10-Q, 8-K, S-1, DEF 14A, Form 4, 13F).
  Real-time within EDGAR's own publication lag (~15 minutes).
- **UK:** Companies House annual filings + significant-control
  changes. Daily refresh.
- **EU:** ESMA-aggregated issuer disclosures. Best-effort coverage,
  varies by jurisdiction.

Not covered:

- Private-company filings outside the above jurisdictions.
- M&A details that haven't reached a public filing yet (rumors are
  in `News`, not here).
- Fund-level positions below 13F reporting thresholds.

Always cite the filing type and date alongside the figure quoted.
"FY revenue $X" alone is insufficient — "FY revenue $X (10-K, FY24,
filed 2025-02-14)" is the citation shape.

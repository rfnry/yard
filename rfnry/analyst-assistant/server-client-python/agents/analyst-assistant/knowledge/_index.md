# Knowledge — Research Reference

Reference material the analyst may cite when interpreting tool
output. Organized by topic; drill into a topic's `_index.md` for
the file list inside.

- [data-sources/_index.md](data-sources/_index.md) — what each
  data source covers, refresh cadence, known gaps
- [methodology/_index.md](methodology/_index.md) — how the
  consultancy frames findings, citation style, missing-data
  handling
- [client-context/_index.md](client-context/_index.md) — what
  belongs in *this* index vs. per-engagement files

Per-engagement knowledge (when an analyst onboards a new client)
lands at the top level of `knowledge/`, not in these subdirs.
Subdir content is shared across all engagements.

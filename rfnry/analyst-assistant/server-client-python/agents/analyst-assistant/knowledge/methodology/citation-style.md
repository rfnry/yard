# Citation Style

Every fact in a research note must trace to a tool call that ran
in the current turn. Cite the tool and the discriminating field:

> Q3 revenue $4.2B (Filings, 10-Q FY25 Q3, filed 2025-11-04).

> Stock down 11% over the last 5 sessions (MarketData, daily,
> 2025-11-03 to 2025-11-07).

> No company-issued statements about the recall in the last 30 days
> (News, search "recall", filtered to issuer press releases).

The citation is tight — name the tool, name the discriminating
field (filing type + date, date range, search query). A reader
should be able to reproduce the call from the citation alone.

# Peer Comparisons

A peer comparison is in scope only when the lead analyst asked for
one explicitly, or when the engagement's setup notes name a peer
set. Otherwise, do not introduce comparators the user didn't ask
about — `AGENT.md` explicitly forbids this ("Compare companies
the lawyer didn't ask about" is a behavior to avoid).

When a comparison **is** in scope:

- Quote the same metric for each peer with the same as-of date.
- Cite each peer's value separately — do not present a combined
  "average" without citing each input.
- If a peer's data is unavailable for the requested as-of date,
  surface the gap; don't substitute a different period.

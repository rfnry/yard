# No speculation

Missing data is a finding. If `/news/{ticker}` returns no items in the
last 30 days, the report says "no recent news on file." It does not
say "no major developments" — you don't know that, you only know what
the tool returned.

Same rule for null fields: if `pe_ratio` is null, the report says
"P/E not reported" — not "unprofitable" and not "reporting issue."

# Report Source by Source

The **Sources** section of every reply lists each tool you might
have called, whether you called it, and what came back in one line.
Even tools you didn't call get a row:

```
- Identity (called: yes, result: "Theodore A. Branham, DOB 1978-03-14, Portland OR")
- CriminalRecords (called: yes, result: "1 record — DUI 2014, convicted")
- CourtRecords (called: no, reason: "case number not provided")
- PropertyRecords (called: yes, result: "1 record — tenant since 2019-05")
- BusinessRegistry (called: no, reason: "no business id provided")
- EmploymentHistory (called: yes, result: "1 record — Pacific Logistics, current")
```

This makes audit trivial: a reviewing lawyer can see at a glance
what was checked and what wasn't, without re-reading the prose.

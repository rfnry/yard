# Missing Is a Finding

If a tool returns 404, an empty list, or "not on file", report it
explicitly under **Summary** and **Sources**. Don't gloss it.

Example **Summary** sentences that surface "missing" well:

- "Public records show no criminal record on file for ID-1024."
- "PropertyRecords returned an empty list — no real estate on file
  in the searched jurisdictions."
- "Court case OR-2014-CRM-99999 was not found in the registry; the
  case number may be wrong or the case is sealed."

Missing data often matters more than present data. Surface it.

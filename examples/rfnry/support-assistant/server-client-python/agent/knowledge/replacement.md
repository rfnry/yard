# Replacement Policy

We ship a replacement under the following conditions:

- **Pre-delivery damage** (warped, broken, missing) — replacement
  ships immediately on photo evidence; no return required (the
  customer keeps or recycles the damaged unit). The order's
  `payment_id` stays captured; we eat the second part's cost.
- **In-warranty defect** within first 90 days of delivery —
  replacement ships immediately; we issue a prepaid return label
  for the original.
- **In-warranty defect** beyond first 90 days — customer ships the
  defective unit back first; replacement ships on receipt.
- **Out-of-warranty** — no replacement. Direct the rep to the
  refurbishment program or a paid replacement quote.

Always confirm the order is real (`Order` returns 200) and the part
is real (`Catalog` returns 200) before promising any replacement.

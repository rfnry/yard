# Follow the Three-Section Shape

Every reply has exactly three sections, in this order:

1. `## Findings` — bulleted facts from tool calls. One fact per line.
   Cite the source in parentheses: `(Catalog PART-12345)`,
   `(Order ORD-100045)`, `(policy: warranty.md)`.
2. `## Recommendation` — one or two short paragraphs the rep can
   relay or act on, grounded in Findings + the policies under
   `knowledge/`.
3. `## Open questions` — bulleted gaps. Empty is valid; if so write
   `none`. If you couldn't get a tool result you needed, name it.

No preamble before `## Findings`. No closing summary. The rep skims
the three sections in a queue.

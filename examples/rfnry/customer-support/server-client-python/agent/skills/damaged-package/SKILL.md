# Damaged Package

When the customer reports a damaged or broken item:

1. Apologize once, briefly.
2. Confirm the order id (per `instructions/confirm-order-id.md`).
3. Ask the customer to send a photo of the damage to
   `support@example.com` with the order id in the subject line.
4. Tell them: a replacement ships once we receive the photo, and they
   keep or recycle the damaged item — they do not need to return it.
5. If the item is over 30 days old, do not promise a replacement; tell
   them you'll connect them with a human (per
   `instructions/escalate-when-unsure.md`).
